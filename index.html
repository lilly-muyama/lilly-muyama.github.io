Interactive Anemia Diagnosis Pathway Plots

A Brief Description

This page displays links to the interactive plots for the paper entitled "Extracting Diagnosis Pathways from Electronic Health Records using Deep Reinforcement Learning". The Sankey diagrams show the pathways learned by the model for each particular anemia class. Each diagram is generated from the trajectories followed by the model leading to that anemia class' diagnosis. The green pathways represent the correctly diagnosed patients whereas the red pathways concluded in an incorrect diagnosis. The orange nodes depict the actions that query for feature values nodes, while the diagnosis actions are the purple nodes at the end. In addition, the size of each flow corresponds to its support, i.e., the number of patients that have it in their pathway. The full article and code can be found here and here respectively.
